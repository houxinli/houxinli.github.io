# Repo Health Check (Static Snapshot)

## Current status
- This repository currently looks like a generated static site snapshot, not a full Hexo source repository.
- `index.html` includes `meta name="generator" content="Hexo 3.8.0"`, which confirms it was generated by Hexo at least once.
- Typical Hexo source files (`package.json`, `_config.yml`, `source/`, `themes/`) are not present in this repository.

## What this means
- You can still publish and update this site as a static website on GitHub Pages.
- Rebuilding old posts via Hexo is currently hard without the original source markdown and theme configs.

## MVP maintainability direction
- Keep this repository as a static deployment repo for now.
- Add feature pages directly (like `/visa-tracker/`) with data files under `/data/`.
- Later, if needed, migrate into a modern static-site source repo (Astro/Eleventy/Hugo), while preserving existing URLs.
